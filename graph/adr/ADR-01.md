id: ADR-01
status: accepted
title: Product charter

Context: prior concept inverted effort into a decorative 2D office view while the
differentiated asset (governance machinery + audited monitoring) was under-invested;
two parallel tracks created coordination overhead at one-sponsor scale.
Decision: found from zero as Engine + App. Engine = governance machinery shipped as
composable skills (ADR-07). App = monitoring/live-ops reference implementation.
Positioning: "governance graph / auditable operating system" — never "knowledge graph"
(crowded space, e.g. Graphify). Office renderer: parked idea, never built here; any
future renderer attaches read-only via STATE-FEED (PRN-07).
Consequences: single plan; src/ grows from zero; carried learnings recorded as PRNs.
Source: sponsor refound sessions, 2026-07-04.
